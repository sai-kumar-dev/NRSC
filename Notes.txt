prepare_data_in_chunks() :
    input vars shapes are logged 
    dimensions are gathered for time, lat, lon
    ocean_mask is used to correctly make it 3D
    total_ocean_points are calculated
    if total_ocean_points > max_samples:
        effective sample_ratio calculated
    else:
        subsampleratio is used 


    reshape data to 2D => (time_dim, -1(lat*lon))

    ocean_mask to 2D
    samples collected to atmost max_samples var
    ocean_mask is used to filter variables to only have ocean points data
    make column stack of variables lists for given points at t
    filter to get only non empty rows where all values are there 

    if the resultant X_t is greater than chunk size randomly select only chunk size indices 

    caluclate windspeed, e_s_sst, e_s_td, fluxes 

    fluxes results are bundled to y

    chunks are appended

    del all variables 

    periodic memory cleanup
    X,y vertical stacks of all chunks
    del chunks
    remove any nan or inf value entries

    returns X,y
